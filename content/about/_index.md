intro:

    Basée à **Angoulême**, PixiCode est une agence d'ingénierie logicielle sur mesure, basée à Angoulême depuis 2023. Nous concevons des sites, applications mobiles et logiciels métiers pour les PME, startups et associations qui veulent posséder leur outil numérique — sans abonnement captif, sans dépendance prestataire.
      
    Notre approche : ne livrer que du code propriétaire, éco-conçu et architecturé pour durer. Ce qu'on ne fait pas est aussi important que ce qu'on fait — pas de Wix, pas de Webflow, pas de projets sans cadrage, pas de maintenance sans contexte

# 2. Section Notre Histoire
history:
  title: "Notre ADN : Souveraineté technique et performance durable"
  text: |
    PixiCode est née d'un constat simple : la plupart des outils numériques livrés aux entreprises ne leur appartiennent pas vraiment. Le prestataire garde les accès. Le code est verrouillé dans une plateforme. Le nom de domaine est enregistré à un tiers.
      
    Nous avons construit PixiCode pour que ça ne soit plus le cas. Chaque projet est architecturé avec une seule contrainte non négociable : à la livraison, l'actif numérique appartient au client. Le code source, les accès, le domaine, la documentation — tout.
      
    Certifiés par l'Institut du Numérique Responsable (INR), nous appliquons les principes d'éco-conception à chaque ligne produite. Agréés CII jusqu'en 2030, nos clients PME bénéficient d'un levier fiscal direct sur leurs investissements de conception
  image: "images/image_team.webp"
