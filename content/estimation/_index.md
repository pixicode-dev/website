## Comment ça marche

Trois étapes, deux minutes, aucune compétence technique requise.

1. **Décrivez votre projet.** Type de projet, ampleur, fonctionnalités clés : quelques questions simples suffisent.
2. **Obtenez votre fourchette.** Une estimation de prix indicative s'affiche immédiatement à l'écran, adaptée à vos réponses.
3. **On affine ensemble.** Nous vous recontactons sous 24 h ouvrées pour préciser le périmètre et, si vous le souhaitez, établir un devis détaillé.

## Nos tarifs de référence

Nos prix sont transparents. Voici les points de départ les plus fréquents. L'estimateur ci-dessus ajuste ces montants à votre projet réel.

### Création de site internet

| Formule | Contenu | À partir de (HT) |
|---|---|---|
| Essentielle | Site vitrine une page | **900 €** |
| Pro | Site 4 pages + formulaire de contact | **3 150 €** |
| Premium | Site 6 pages avec blog | **4 080 €** |
| Sur mesure | CMS autonome (Strapi), application web, projet spécifique | **dès 4 800 €** |

### Site de collectivité

Site de commune, EPCI ou office de tourisme : **à partir de 5 760 € HT**, avec accessibilité RGAA en option, éco-conception et réversibilité incluses.

### Application mobile & logiciel métier

Ces projets sont sur mesure par nature. Nous les estimons à partir du temps de développement prévisible, sur la base d'un **taux journalier de 480 € HT**. L'estimateur vous donne une fourchette selon le périmètre décrit.

### Hébergement & infogérance

À partir de **15 €/mois HT** pour un site statique, **25 €/mois** dès qu'un formulaire est en production, **55 €/mois** pour un site à CMS autonome. Gammes dédiées pour WordPress et pour les collectivités.

> Tous les prix s'entendent hors taxes. Les créations incluent l'intégration responsive, l'éco-conception (référentiel INR), le socle SEO technique, la mise en ligne et une garantie corrective de 3 mois.
